# /promo — Promotion Campaign Plan for salon-flow CRM

Agent for designing promotional campaigns to acquire and retain salon-flow paying customers.
Audience: beauty salon owners, administrators, independent masters.

---

## Phase 1: Goal & Context

Ask the user:
1. **Campaign goal?**
   - Acquire new trial users?
   - Convert existing trials to paid?
   - Reactivate churned salons?
   - Upsell current customers to a higher plan?
   - Launch in a new city or region?
2. **Budget?** (paid ads, sponsorships — or zero-budget only)
3. **Timeline?** (one-time push, or recurring campaign)
4. **Current bottleneck?** (not enough leads / trials don't convert / churn too high)

---

## Phase 2: Campaign Format Selection


| Goal | Recommended Format |
|------|--------------------|
| New trials | Free trial extension offer + demo call — targeted at salon owners in Telegram groups |
| Trial → paid | Urgency campaign: "Your trial ends in 3 days — lock in [discount]%" |
| Churn reactivation | Winback: "We've added [feature] — come back and try it free for 2 weeks" |
| Upsell | Feature spotlight: show ROI of premium tier with real salon data |
| New region launch | Local outreach: partner with regional beauty industry community |
| Brand awareness | Case study content: "How [Salon Name] grew repeat bookings 40% with salon-flow" |

## Phase 4: Message Templates

**Telegram / DM outreach (cold):**
```
Привет [имя]! Веду CRM для салонов красоты — salon-flow.
Помогает автоматизировать запись, вести клиентскую базу и программу лояльности.
Для новых салонов — первый месяц бесплатно.
Можно 15 минут созвониться, покажу как это работает?
```

**Trial → Paid urgency:**
```
[Имя], твой пробный период заканчивается [дата].
За это время [N] клиентов записались через salon-flow.
Подключись сейчас — [скидка]% на первые 3 месяца, только до [дата].
[ссылка на оплату]
```

**Winback (бывший клиент):**
```
[Имя], привет! Ты пробовал salon-flow [N] месяцев назад.
С тех пор мы добавили [фича 1] и [фича 2].
Хочу дать тебе 2 недели бесплатно — посмотри, что изменилось.
Интересно? Отвечай сюда.
```

**Instagram Reel / caption:**
```
[Хук: "Как салон перестал терять клиентов после первого визита"]
[Контент: короткая история / цифра / результат реального клиента]
[CTA: "Ссылка в bio — попробуй бесплатно 30 дней"]
```
