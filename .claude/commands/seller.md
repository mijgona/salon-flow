# /seller — Sales Plan for salon-flow CRM

Agent for building a sales plan to grow paying customers for the salon-flow product.
Target buyer: beauty salon owners and administrators.

---

## Phase 1: Current State Audit

Ask the user to provide current numbers:

1. **Product stage** — MVP / beta / launched / scaling?
2. **Current customers** — how many paying salons, how many on trial/free?
3. **Pricing** — monthly subscription cost, tiers if any
4. **Active acquisition channels** — where do leads come from today?
5. **Conversion** — leads per month → trials → paid (even rough estimates)
6. **Churn** — how many salons cancel per month?

If no data yet, skip to Phase 3 and build a plan from zero.

## Phase 4: Weekly Sales Rhythm

```
Weekly Checklist:
- Mon: review new trials from last week, assign follow-up
- Tue: direct outreach — contact [N] salon owners via DM or call
- Wed: follow up trials day 7 — offer onboarding help
- Thu: content post (Instagram/Telegram) targeting salon owners
- Fri: review closed deals and lost deals — update channel scorecard
```
